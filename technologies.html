2.1 HTML5 — язык разметки

HTML (HyperText Markup Language) — стандартный язык разметки для создания веб-страниц. Версия HTML5 была принята в 2014 году и стала главным стандартом.

HTML определяет структуру страницы: заголовки, абзацы, списки, изображения, ссылки, таблицы и формы. Каждый элемент описывается тегами.

Базовая структура HTML-документа:

HTML
<!-- Объявление типа документа -->
<!DOCTYPE html>
<html lang="ru">

<head>
  <!-- Мета-информация -->
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Название страницы</title>
  <link rel="stylesheet" href="style.css" />
</head>

<body>
  <!-- Видимое содержимое страницы -->
  <header>
    <h1>Заголовок сайта</h1>
    <nav>...</nav>
  </header>

  <main>
    <section>
      <h2>Раздел</h2>
      <p>Текстовый контент.</p>
    </section>
  </main>

  <footer>Подвал сайта</footer>
  <script src="main.js"></script>
</body>
</html>

Семантические теги HTML5:

HTML5 ввёл семантические теги, которые описывают смысл содержимого, а не только его внешний вид. Это улучшает доступность и SEO.

2.2 CSS3 — каскадные таблицы стилей

CSS (Cascading Style Sheets) определяет внешний вид HTML-элементов: цвета, шрифты, отступы, размеры, анимации и адаптивность. CSS3 — актуальная версия стандарта.

Пример стилизации навигации:

CSS
/* CSS-переменные (кастомные свойства) */
:root {
  --color-primary: #4f9cf9;
  --font-main: 'Onest', sans-serif;
  --radius: 12px;
}

/* Навигационная панель */
.navbar {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 0 2rem;
  background: #0a0d14;
  height: 64px;
  position: fixed;
  top: 0;
  width: 100%;
}

/* Адаптивность для мобильных */
@media (max-width: 680px) {
  .navbar {
    padding: 0 1rem;
  }
}

Ключевые возможности CSS3:

  • Flexbox — гибкая однострочная/многострочная раскладка элементов
  • CSS Grid — двумерная сетка для сложных макетов
  • Media Queries — адаптивность под разные экраны
  • CSS-переменные — хранение и переиспользование значений
  • Анимации и transitions — плавные изменения свойств
  • CSS-фильтры и трансформации — визуальные эффекты

2.3 JavaScript — язык программирования

JavaScript — единственный язык программирования, нативно поддерживаемый браузерами. Он обеспечивает интерактивность: реакцию на действия пользователя, динамическое изменение страницы, работу с формами и запросы к серверу.

Пример: обработка формы обратной связи:

JavaScript
// Получаем элементы формы
const form = document.querySelector('#contactForm');
const nameInput = document.querySelector('#name');

// Слушаем событие отправки формы
form.addEventListener('submit', function(event) {
  event.preventDefault(); // отменяем стандартное поведение

  const name = nameInput.value.trim();

  if (name === '') {
    alert('Пожалуйста, введите ваше имя');
    return;
  }

  // Динамически добавляем сообщение
  const msg = document.createElement('p');
  msg.textContent = `Спасибо, ${name}! Мы свяжемся с вами.`;
  form.replaceWith(msg);
});

Основные концепции JavaScript:

🔗DOM

Document Object Model — программный интерфейс для работы с HTML-документом.

📡Events

Обработка событий: клик, наведение, ввод, прокрутка, загрузка страницы.

⚡Fetch API

Асинхронные запросы к серверу без перезагрузки страницы (AJAX).

📦ES6+

Современный синтаксис: стрелочные функции, деструктуризация, async/await.

2.4 Взаимодействие HTML, CSS и JavaScript

Три технологии работают в связке. HTML задаёт структуру, CSS оформляет внешний вид, JavaScript добавляет поведение.

🔄 Принцип разделения ответственности: HTML, CSS и JS должны быть разделены в разные файлы. Это упрощает сопровождение кода, ускоряет загрузку через кэширование и позволяет нескольким разработчикам работать параллельно.
